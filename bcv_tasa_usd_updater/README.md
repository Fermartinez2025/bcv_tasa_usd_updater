# BCV Tasa USD Updater

> **Nota**: El README principal y la documentación actualizada se encuentran en la raíz del proyecto: [README.md](../README.md).

Este directorio contiene una copia local de los recursos de base de datos y scripts de prueba del proyecto **BCV Tasa USD Updater**.

###  Características del Proyecto
-    **Scraping**: Extracción oficial del portal BCV.
-    **Compatibilidad**: Soporte para SQL Server 2000 y 2019.
-    **Inteligencia**: Gestión de feriados y cierres bancarios.
-    **Alertas**: Notificaciones HTML vía SMTP.

Para instrucciones de instalación y configuración, por favor consulte el [README principal en la raíz](../../README.md).
